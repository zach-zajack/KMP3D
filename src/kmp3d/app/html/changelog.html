KMP3D 7.0 Changelog

Cameras

  • Under the Cameras section, you can now select to preview opening cameras and replay cameras.
  • Added ability to edit relative view start positions, and corrected various options for each camera type.
  • Camera indices are now editable, also under the camera preview section.

KMP Merging

  • If overwriting a file, all blank sections will be preserved from the previous file.

Visibility

  • Related sections (e.g. checkpoints and respawns) can now be previewed easily by checking the boxes beneath the sections list.
  • Conditional Out of Bounds can now be previewed.
  • The current selected group will be a different color to other groups.
  • Some KCL flags are no longer imported to improve visibility.

Accuracy

  • All 32 Nintendo KMP files now compile 1:1, with exception to DS Peach Gardens due to an incorrect direct graph setup for the ENPH.
  • STGI flare information is preserved.

Misc. bugfixes

  • External settings now properly save.
  • KMPs now import faster.
  • Unsigned integers can be set to -1 for route IDs.
  • Group headers now export correctly.
